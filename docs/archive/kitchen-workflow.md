# Kitchen Workflow System — Design Specification

> **⚠️ DEPRECATED — Out of Scope for v1**
>
> **Why:** Kitchen Display System (KDS) screens are not practical in Myanmar coffee shop environments — heat, water, steam, dust, and power instability damage screens. The standard workflow is POS → Kitchen Thermal Printer.
>
> **Replacement:** Kitchen printer routing is handled via `printer_integration` (Growth tier). See `docs/specs/inventory-model.md` for the simplified inventory model. The `kitchen_display` DB key is dead code kept for forward compatibility only.
